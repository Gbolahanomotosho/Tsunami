# Tsunami



  Tsunami is a very powerful UDP flooding tool which uses SOCK_DGRAM

  It send User Datagram Protocol packet to the target ip address

  Since UDP is connectionless, sessionless and does not require a three way hansdshake connection

  This makes UDP vulnerable to flooding 

  UDP flooding can flood almost every network that uses UDP network by trying random port 

  Till it hits the right port since it doesnt require any specific port to flood
 
  It can flood PC, Mobile, Smart watch, Printer, Game server, Smart tv, Router, Wifi, IoT device and so on.
  
   

  You can test your network or server strength with it.
